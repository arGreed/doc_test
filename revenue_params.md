# Показатели выручки
## Выручка по услугам
### Альтернативные наименования
-	Фактическая выручка, руб;
-	Выручка;
-	Revenue;
### Формула расчёта
$$ R_{\text{услуг}} = \sum_{i=1}^{n} r_i $$
*где $n$ — совокупное количество оказанных услуг, $r_i$ — объём денежных средств, полученных с $i$-й услуги.*
### Краткое описание параметра
__Выручка по услугам__ - Суммарная выручка по оказанным услугам
