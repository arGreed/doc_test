## <a id="head-1"></a>Выручка по услугам
### <a id="alt-names-1"></a>Альтернативные наименования
- Фактическая выручка, руб;
- Выручка;
- Revenue;

### <a id="desc-1"></a>Краткое описание параметра
**Выручка по услугам** — Суммарная выручка по оказанным услугам

---

## <a id="head-2"></a>Выручка предыдущий месяц, руб
### <a id="desc-2"></a>Краткое описание параметра
**Выручка предыдущий месяц, руб** — Суммарная выручка по оказанным услугам предыдущего месяца за период, аналогичный выбранному

### <a id="prim-2"></a>Примечания
Также используется для вычисления показателя MTM

---

## <a id="head-3"></a>Выручка прошлого года, руб
### <a id="desc-3"></a>Краткое описание параметра
**Выручка прошлого года, руб** — Суммарная выручка по оказанным услугам прошлого года за период, аналогичный выбранному

### <a id="prim-3"></a>Примечания
Также используется для вычисления показателя YTY

---

## <a id="head-4"></a>Выручка по оплаченным услугам, руб
### <a id="desc-4"></a>Краткое описание параметра
**Выручка по оплаченным услугам, руб** — Суммарная выручка по оказанным услугам, по которым произошла фактическая оплата

### <a id="prim-4"></a>Примечания
Используется в случаях, когда есть заметная разница во времени между выставлением услуги и ее оплатой (например, при госпитализации)

---

## <a id="head-5"></a>Выручка новых пациентов
### <a id="desc-5"></a>Краткое описание параметра
**Выручка новых пациентов** — Суммарная выручка по оказанным услугам для новых пациентов. При этом новым пациент считается только при своем первом визите

---

## <a id="head-6"></a>% выручки новых, руб
### <a id="desc-6"></a>Краткое описание параметра
**% выручки новых, руб** — представляет собой процентную долю выручки, приходящуюся на «новых» пациентов

---

## <a id="head-7"></a>Выручка по первичным пациентам
### <a id="desc-7"></a>Краткое описание параметра
**Выручка по первичным пациентам** — Суммарная выручка по оказанным услугам для первичных пациентов (интервал первичности — 1, 3, 6 или 12 месяцев)

### <a id="prim-7"></a>Примечания
Первичный пациент — пациент, который не был в клинике в течение всего периода первичности

---

## <a id="head-8"></a>Среднедневная выручка
### <a id="desc-8"></a>Краткое описание параметра
**Среднедневная выручка** — Средняя выручка по оказанным услугам в выбранном периоде за 1 день

### <a id="prim-8"></a>Примечания
Учитываются только дни, где есть хотя бы 1 услуга

---

## <a id="head-9"></a>Среднемесячная выручка
### <a id="desc-9"></a>Краткое описание параметра
**Среднемесячная выручка** — Средняя выручка по оказанным услугам за 1 месяц. Под 1 месяцем понимается 365 дней/12, т.е. ~30,42 дней

---

## <a id="head-10"></a>Выручка в кабинетах
### <a id="desc-10"></a>Краткое описание параметра
**Выручка в кабинетах** — Суммарная выручка по услугам, оказанных пациентам в амбулаторных кабинетах

### <a id="prim-10"></a>Примечания
Рассчитывается при учете клиникой расписания по кабинетам приема

---

## <a id="head-11"></a>Выручка по оплатам (Сумма оплат)
### <a id="desc-11"></a>Краткое описание параметра
**Выручка по оплатам (Сумма оплат)** — Сумма оплат, произведенных за период

### <a id="prim-11"></a>Примечания
В отличие от выручки по услугам собирается по финансовым транзакциям. Оплаты не всегда можно отнести к конкретной услуге или к конкретному врачу

---

## <a id="head-12"></a>Среднедневная выручка по оплатам
### <a id="desc-12"></a>Краткое описание параметра
**Среднедневная выручка по оплатам** — Средняя выручка по платежам за 1 день

---

## <a id="head-13"></a>Среднемесячная выручка по оплатам
### <a id="desc-13"></a>Краткое описание параметра
**Среднемесячная выручка по оплатам** — Средняя выручка по платежам за 1 месяц. Под 1 месяцем понимается 365 дней/12, т.е. ~30,42 дней

---

## <a id="head-14"></a>Скидка, руб
### <a id="desc-14"></a>Краткое описание параметра
**Скидка, руб** — скидка в рублях

---

## <a id="head-15"></a>Скидка, %
### <a id="desc-15"></a>Краткое описание параметра
**Скидка, %** — скидка в % от прайса

---
