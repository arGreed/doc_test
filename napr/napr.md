## 📬 Базовые метрики направлений

### <a id="head-1"></a>📤 # направлений

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Количество выписанных направлений |
| Referrals Count |
| Assigned Referrals |

**📝 Описание:**  
Общее количество выписанных направлений пациентам за выбранный период

> 📌 **Определение:** Направление = документ/запись в МИС с назначением услуг, исследований или консультаций
>
> 🎯 **Интерпретация:** Индикатор врачебной активности по назначению дополнительных услуг

---

### <a id="head-2"></a>✅ # реализованных в периоде направлений

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Исполненные направления |
| Fulfilled Referrals |
| Completed Referrals |

**📝 Описание:**  
Количество направлений, которые были исполнены (реализованы) в выбранном периоде

> 🧮 **Критерий:** Направление считается реализованным, если хотя бы одна услуга из него оказана пациенту
>
> 📌 **Особенность:** Считается по дате **оказания услуги**, а не по дате выписки направления

---

### <a id="head-3"></a>📝 # назначенных направлений

**📝 Описание:**  
Количество направлений, выписанных пациентам в выбранном периоде

> 🧮 **Критерий:** Учитываются направления по дате **создания/назначения**
>
> 📌 **Примечание:** Может отличаться от «# направлений», если в системе раздельно учитываются черновики и утверждённые назначения

---

## 💰 Финансовые метрики направлений

### <a id="head-4"></a>💵 Выручка назначенных направлений, руб

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Потенциальная выручка направлений |
| Potential Referral Revenue |

**📝 Описание:**  
Суммарная выручка по услугам, указанным в выписанных направлениях

> 🧮 **Расчёт:** Сумма прайсовых стоимостей услуг, включённых в направления
>
> 🎯 **Интерпретация:** Показывает финансовый потенциал назначений — сколько клиника могла бы заработать при 100% исполнении

---

### <a id="head-5"></a>💰 Выручка реализованных в периоде направлений, руб

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Факт выручка по исполненным направлениям |
| Actual Referral Revenue |

**📝 Описание:**  
Суммарная выручка по услугам в реализованных (исполненных) направлениях

> 🧮 **Особенность расчёта:** Считается по дате **прихода пациента** (оказания услуги)
>
> 📌 **Важно:** Направления могли быть назначены за пределами анализируемого периода — учитывается только факт исполнения в периоде

---

### <a id="head-6"></a>📅 Выручка реализованных направлений в периоде назначения, руб

**📝 Описание:**  
Суммарная выручка по услугам в направлениях, которые были **и назначены, и реализованы** в выбранном периоде

> 🧮 **Критерий:** Учитываются только направления, у которых:
> - Дата назначения ∈ выбранный период
> - Дата исполнения (оказания услуги) ∈ выбранный период
>
> 🎯 **Интерпретация:** Показывает эффективность «быстрого» исполнения назначений в рамках одного периода

---

### <a id="head-7"></a>👥 # пациентов по реализованным направлениям

**📝 Описание:**  
Количество уникальных пациентов, направления которых были исполнены в выбранном периоде

> 📌 **Примечание:** Один пациент может иметь несколько реализованных направлений — учитывается как один уникальный пациент

---

### <a id="head-8"></a>📊 Доля реализованных в периоде направлений

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Конверсия направлений |
| Referral Fulfillment Rate |
| % исполнения направлений |

**📝 Описание:**  
Доля реализованных направлений от общего количества выписанных направлений

> 🧮 **Расчёт:** `# реализованных в периоде направлений / # направлений * 100`
>
> 🎯 **Интерпретация:** 
> - Высокий % → пациенты следуют рекомендациям, эффективная коммуникация
> - Низкий % → возможные барьеры: цена, доступность, непонимание важности

## 🧮 Средние значения и аналитика

### <a id="head-9"></a>🧮 Средний чек по направлениям

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Avg Check per Referral |
| Средняя стоимость направления |

**📝 Описание:**  
Средняя потенциальная выручка на одно выписанное направление

> 🧮 **Расчёт:** `Потенциальная выручка направлений / # назначенных направлений`
>
> 📌 **Определение:** Потенциальная выручка = сумма услуг, на которые выставлены направления (по прайсу)
>
> 🎯 **Интерпретация:** Показывает «вес» назначений: комплексные программы дают высокий чек, разовые консультации — низкий

---

### <a id="head-10"></a>🧮 Средний чек по исполненным направлениям

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Avg Check per Fulfilled Referral |
| Фактический чек направления |

**📝 Описание:**  
Средняя фактическая выручка на одно исполненное направление

> 🧮 **Расчёт:** `Выручка по исполненным направлениям / # исполненных направлений`
>
> 🎯 **Интерпретация:** 
> - Сравнение с «Средним чеком по направлениям» показывает, насколько фактическое исполнение соответствует назначениям
> - Разница может возникать из-за скидок, замены услуг, частичного исполнения

---

### <a id="head-11"></a>👨‍⚕️ Всего исполнено направлений от врача на сумму

**📝 Описание:**  
Выручка по исполненным направлениям, выписанным врачом, за период (независимо от даты создания направления)

> 🧮 **Критерий:** Учитываются все направления от врача, услуги по которым оказаны в выбранном периоде
>
> ⚙️ **Фильтрация:** Возможна разбивка по типам направлений (на анализы, на консультации, на процедуры и т.д.)
>
> 🎯 **Интерпретация:** Показывает вклад врача в генерацию выручки через назначения

---

### <a id="head-12"></a>👨‍⚕️ Исполнено своих направлений врачом на сумму

**📝 Описание:**  
Выручка по исполненным направлениям, которые врач выписал **и сам же исполнил** (оказал услугу лично)

> 🧮 **Критерий:** 
> - Направление выписано врачом А
> - Услуга оказана тем же врачом А
> - Дата оказания услуги ∈ выбранному периоду
>
> 🎯 **Интерпретация:** 
> - Высокое значение → врач работает с собственным потоком назначений
> - Низкое значение → направления передаются коллегам или внешним исполнителям

## 🏥 Планы лечения и курсы (ПЛ)

> 📌 **Определение:** План лечения (ПЛ) / курс = структурированный набор услуг, назначенных пациенту для достижения терапевтической цели (например, курс физиотерапии, программа реабилитации, комплекс анализов)

---

### <a id="head-13"></a>📋 # пациентов ПЛ/курсов

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Пациенты с планами лечения |
| Patients with Treatment Plans |

**📝 Описание:**  
Количество пациентов, которым были назначены планы лечения или курсы в выбранном периоде

> 📌 **Примечание:** Учитываются уникальные пациенты, даже если им назначено несколько ПЛ

---

### <a id="head-14"></a>📊 % пациентов с ПЛ/курсами

**📝 Описание:**  
Доля пациентов, которым назначены планы лечения/курсы, от общего количества пациентов

> 🧮 **Расчёт:** `# пациентов с ПЛ / # пациентов * 100`
>
> 🎯 **Интерпретация:** 
> - Высокий % → клиника активно использует комплексные программы ведения
> - Низкий % → преобладают разовые обращения, возможность для развития программ

---

### <a id="head-15"></a>✅ # пациентов исп. ПЛ/курсов

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Пациенты с исполненными ПЛ |
| Patients with Completed Plans |

**📝 Описание:**  
Количество пациентов, у которых планы лечения/курсы, созданные за период, были исполнены (полностью или частично)

> 🧮 **Критерий:** Учитываются ПЛ, созданные в выбранном периоде, по которым оказана хотя бы одна услуга

---

| Название | 
|----------|
| Потенциальная выручка ПЛ |
| Potential Treatment Plan Revenue |

**📝 Описание:**  
Выручка по назначенным планам лечения/курсам (потенциальная, по прайсу)

> 🧮 **Расчёт:** Сумма прайсовых стоимостей услуг, включённых в созданные ПЛ
>
> 🎯 **Интерпретация:** Финансовый потенциал программ лечения на этапе назначения

---

### <a id="head-17"></a>💰 Выручка исп. ПЛ/курсов

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Факт выручка по исполненным ПЛ |
| Actual Treatment Plan Revenue |

**📝 Описание:**  
Выручка по фактически оказанным услугам в рамках планов лечения/курсов, созданных за период

> 🧮 **Критерий:** Учитываются только услуги, оказанные в рамках ПЛ, созданных в выбранном периоде
>
> 🎯 **Интерпретация:** Реальный финансовый результат от программ лечения

---

### <a id="head-18"></a>📈 % исполнения ПЛ/курсов

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Конверсия ПЛ |
| Treatment Plan Fulfillment Rate |
| % завершения курсов |

**📝 Описание:**  
Доля исполненных услуг в рамках созданных за период планов лечения/курсов

> 🧮 **Расчёт:** `Сумма оплат по исполненным курсам / Сумма оплат по назначенным курсам * 100`
>
> 🎯 **Интерпретация:** 
> - Высокий % → пациенты завершают программы, эффективное ведение
> - Низкий % → барьеры исполнения: стоимость, время, мотивация, доступность

---

### <a id="head-19"></a>👨‍⚕️ Всего исполнено ПЛ/курсов от врача за период на сумму

**📝 Описание:**  
Выручка по исполненным услугам в планах лечения/курсах, выписанных врачом, за период (независимо от даты создания ПЛ)

> 🧮 **Критерий:** Учитываются все ПЛ от врача, услуги по которым оказаны в выбранном периоде
>
> 🎯 **Интерпретация:** Вклад врача в реализацию комплексных программ

---

### <a id="head-20"></a>👨‍⚕️ Исполнено своих ПЛ/курсов врачом на сумму

**📝 Описание:**  
Выручка по исполненным услугам в ПЛ/курсах, которые врач выписал **и сам же исполнил**

> 🧮 **Критерий:** 
> - ПЛ выписан врачом А
> - Услуга в рамках ПЛ оказана тем же врачом А
> - Дата оказания ∈ выбранному периоду
>
> 🎯 **Интерпретация:** Показывает, насколько врач ведёт пациента по программе лично

---

### <a id="head-21"></a>📊 Доля исполненных ПЛ/курсов

**📋 Альтернативные наименования:**

| Название | 
|----------|
| % завершённых планов |

**📝 Описание:**  
Доля пациентов, у которых планы лечения/курсы были исполнены, от общего количества пациентов с назначенными ПЛ

> 🧮 **Расчёт:** `# пациентов исп. ПЛ / # пациентов ПЛ * 100`
>
> 🎯 **Интерпретация:** Ключевой показатель эффективности программ лечения и удержания пациентов в процессе терапии

> 💡 **Рекомендации по использованию:**
> - **Доля реализованных направлений** — ключевой индикатор коммуникации: низкое значение может указывать на необходимость улучшения объяснения важности назначений
> - **Сравнение «потенциальной» и «фактической» выручки** по направлениям помогает оценить потери из-за неисполнения назначений
> - **% исполнения ПЛ/курсов** — метрика качества ведения пациента: используйте её для оценки эффективности программ и работы кураторов
> - **«Свои направления/ПЛ»** показывают, насколько врач замыкает поток на себе — важно для анализа нагрузки и перекрёстных назначений
> - При анализе **выручки реализованных направлений** учитывайте лаг между назначением и исполнением: направления, выписанные в конце месяца, часто исполняются в следующем

---

> 🔐 **Требования к данным:**
> - В МИС должна быть реализована сущность «Направление» с возможностью связи «назначение → исполнение»
> - Для планов лечения (ПЛ) требуется структурированное хранение: список услуг, статусы исполнения, привязка к пациенту и врачу
> - Корректная датировка: дата назначения направления и дата оказания услуги должны фиксироваться раздельно
> - Для расчёта «своих направлений/ПЛ» необходима однозначная связь «врач-назначитель → врач-исполнитель»

---

> ⚙️ **Настройки и кастомизация:**
> - Типы направлений (на анализы, консультации, процедуры) настраиваются в справочнике — это позволяет сегментировать аналитику
> - Критерий «исполнения» направления (полное/частичное) может настраиваться под процессы клиники
> - Для ПЛ можно настроить порог «завершённости» (например, ≥80% услуг исполнено = курс завершён)
