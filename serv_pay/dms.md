## <a id="head-1"></a>💰 Выручка факт ДМС+ЮЛ

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Факт выручка ДМС+ЮЛ |
| Revenue DMS+LE Actual |

**📝 Описание:**  
Суммарная фактическая выручка от Добровольного медицинского страхования (ДМС) и Юридических Лиц (ЮЛ)

> 🧮 **Расчёт:** `Факт выручка ДМС + Факт выручка ЮЛ`
>
> ⚠️ **Примечание:** Доступно только на кастомизированных дашбордах

## <a id="head-9"></a>📈 Кол-во пациентов с ДМС, YTY

**📝 Описание:**  
Процентное изменение количества уникальных пациентов с ДМС по годам (Year-To-Year)

> 🧮 **Расчёт:** `100 * (Пациентов ДМС - Пациентов ДМС в предыдущем году) / Пациентов ДМС в предыдущем году`
>
> ⚙️ **Фильтр:** `TypeOfContractor = 'СК'`
>
> ⚠️ **Примечание:** Доступно только на кастомизированных дашбордах

## <a id="head-11"></a>📊 % ДМС от всех пациентов

**📝 Описание:**  
Процент пациентов, которые имеют ДМС, относительно общего числа пациентов

> 🧮 **Расчёт:** `100 * Пациентов ДМС / Пациентов`
>
> 🎯 **Интерпретация:** Показывает долю пациентов со страховым покрытием в общем потоке
>
> ⚠️ **Примечание:** Доступно только на кастомизированных дашбордах

---

## <a id="head-12"></a>💰 Выручка по ДМС, YTY%

**📝 Описание:**  
Процентное изменение выручки по ДМС по годам (Year-To-Year)

> 🧮 **Расчёт:** `100 * (Выручка ДМС - Выручка ДМС прошлого года) / Выручка ДМС прошлого года`
>
> ⚙️ **Фильтр:** `TypeOfContractor = 'СК'`
>
> ⚠️ **Примечание:** Доступно только на кастомизированных дашбордах

## <a id="head-13"></a>📈 Доля выручки по ДМС, %

**📋 Альтернативные наименования:**

| Название | 
|----------|
| % выручки ДМС |
| VHI Revenue Share |

**📝 Описание:**  
Процентное соотношение выручки, полученной от ДМС, к общей выручке клиники

> 🧮 **Расчёт:** `100 * Выручка ДМС / Общая выручка`
>
> 🎯 **Интерпретация:** Показывает зависимость финансового результата от страховых партнёров
>
> ⚠️ **Примечание:** Доступно только на кастомизированных дашбордах

---

## <a id="head-14"></a>📊 Доля выручки по ДМС, YTY%

**📝 Описание:**  
Процентное изменение доли выручки от ДМС между текущим и предыдущим годом

> 🧮 **Расчёт:** `(Доля выручки ДМС текущий год - Доля выручки ДМС прошлый год)`
>
> 🎯 **Интерпретация:** 
> - Положительное значение → рост зависимости от ДМС
> - Отрицательное значение → диверсификация источников выручки
>
> ⚠️ **Примечание:** Доступно только на кастомизированных дашбордах

---

## <a id="head-15"></a>📈 % ДМС от всех пациентов, YTY

**📝 Описание:**  
Изменение доли пациентов с ДМС от общего числа пациентов между текущим и предыдущим годом (Year-To-Year)

> 🧮 **Расчёт:** `% ДМС от всех пациентов (текущий год) - % ДМС от всех пациентов (прошлый год)`
>
> 🎯 **Интерпретация:** Показывает динамику привлечения пациентов со страховым покрытием
>
> ⚠️ **Примечание:** Доступно только на кастомизированных дашбордах

> 💡 **Рекомендации по использованию:**
> - **Фильтр TypeOfContractor = 'СК'** применяется для метрик, связанных исключительно со страховыми компаниями
> - **Доля выручки по ДМС** — ключевой показатель финансовой устойчивости: высокая зависимость от одного канала несёт риски
> - При анализе **динамики (YTY)** учитывайте сезонность заключения корпоративных договоров (часто — начало года)
> - Метрики **плана и факта** позволяют оценивать исполнение обязательств перед контрагентами
> - Все метрики раздела требуют **корректной разметки контрагентов** в МИС и настройки интеграции

---

> 🔐 **Требования к доступу:**
> - Раздел доступен только на кастомизированных дашбордах
> - Необходима настройка справочника контрагентов с типами: `СК` (страховая компания), `ЮЛ` (юридическое лицо)
> - Для расчёта долей требуется корректная агрегация данных по типам оплат и договорам
