## <a id="head-1"></a>📅 # посещений в прошлом году

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Посещения прошлый год |
| Visits PY |

**📝 Описание:**  
Общее количество уникальных кодов обращений (посещений) пациентов в прошлом году за период, аналогичный выбранному

> 📌 **Важно:** Учитываются только уникальные коды посещений из МИС

---

## <a id="head-2"></a>🎯 План посещений

**📝 Описание:**  
Общее запланированное количество посещений

> ⚙️ **Особенность:** В план включаются часы, которые относятся к `visit_count`
>
> ⚠️ **Примечание:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-3"></a>🔗 # визитов

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Количество визитов |
| Visits Count |

**📝 Описание:**  
Количество визитов пациента — все услуги, оказанные в один день, считаются одним визитом

> 🧮 **Определение:** `1 визит = все услуги врачей в одну дату`
>
> 📌 **Примечание:** Позволяет оценить частоту обращений, а не количество оказанных услуг

---

## <a id="head-4"></a>⏱️ Посещений в час

**📝 Описание:**  
Среднее количество посещений в час — показатель производительности работы врача

> 🧮 **Расчёт:** `# посещений / # часов в фактическом расписании (по дошедшим пациентам)`
>
> 🎯 **Интерпретация:** Чем выше значение, тем эффективнее используется рабочее время врача

---

## <a id="head-5"></a>👤 Посещений на пациента в прошлом году

**📝 Описание:**  
Среднее количество посещений пациентом клиники за выбранный период в предыдущем году

> 🧮 **Расчёт:** `# посещений в прошлом году / # пациентов в предыдущем году`
>
> 📌 **Примечание:** Показывает частоту обращений пациентов в историческом разрезе

---

## <a id="head-6"></a>📈 Посещений на пациента с начала года

**📝 Описание:**  
Среднее количество посещений пациентом клиники с начала текущего года

> 🧮 **Расчёт:** `# посещений с начала года / # пациентов с начала года`
>
> 🎯 **Интерпретация:** Индикатор вовлечённости пациентов в текущем году

---

## <a id="head-7"></a>📊 Посещений на пациента, YTY

**📝 Описание:**  
Динамика показателя «Посещений на пациента» к предыдущему году (Year-To-Year)

> 🧮 **Расчёт:** `Посещений на пациента в этом году / Посещений на пациента в предыдущем году - 1`
>
> 📌 **Примечание:** Положительное значение указывает на рост частоты обращений

---

## <a id="head-8"></a>👨‍⚕️ Посещений на врача

**📝 Описание:**  
Среднее количество посещений, приходящихся на одного врача

> 🧮 **Расчёт:** `Посещений / Врачи`
>
> 🎯 **Интерпретация:** Показатель загрузки врачебного состава

---

## <a id="head-9"></a>✅ Исполнение плана по посещениям, %

**📝 Описание:**  
Процентное отклонение фактического количества посещений от планового

> 🧮 **Расчёт:** `100 * (Факт посещений / План посещений - 1)`
>
> ⚠️ **Примечание:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-10"></a>⚖️ Отклонение от плана посещений

**📋 Альтернативные наименования:**

| Название | 
|----------|
| ∆ посещений к плану |
| Разница посещений план/факт |

**📝 Описание:**  
Абсолютная разница между фактическим и запланированным количеством посещений

> 🧮 **Расчёт:** `Факт посещений - План посещений`
>
> ⚠️ **Примечание:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-11"></a>📉 ∆ посещений к плану

**📝 Описание:**  
Разница между фактическим и плановым количеством посещений в абсолютном выражении

> 🧮 **Расчёт:** `# посещений - посещений план`
>
> ⚠️ **Примечание:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-12"></a>📈 ∆ посещений к плану %

**📝 Описание:**  
Процентное отклонение фактических посещений от плановых

> 🧮 **Расчёт:** `100 * (# посещений - посещений план) / посещений план`
>
> ⚠️ **Примечание:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-13"></a>📊 Посещения, YTY%

**📝 Описание:**  
Процентное изменение количества посещений к предыдущему году (Year-To-Year)

> 🧮 **Расчёт:** `100 * (Посещения - Посещения в предыдущем году) / Посещения в предыдущем году`

---

## <a id="head-14"></a>📅 Посещения, прошлый год

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Посещения PY |
| Visits Previous Year |

**📝 Описание:**  
Общее количество посещений, зарегистрированных в предыдущем году за аналогичный период

> 📌 **Примечание:** Базовая метрика для расчёта годовой динамики

---

## <a id="head-15"></a>📉 Посещения, YTY

**📝 Описание:**  
Абсолютное изменение количества посещений сравнительно с предыдущим годом

> 🧮 **Расчёт:** `Посещения - Посещения в предыдущем году`

---

## <a id="head-16"></a>📅 Посещения, MTM%

**📝 Описание:**  
Процентное изменение количества посещений от месяца к месяцу (Month-To-Month)

> 🧮 **Расчёт:** `100 * (Посещений - Посещений в предыдущем месяце) / Посещений в предыдущем месяце`

---

## <a id="head-17"></a>🆕 # посещений новых

**📝 Описание:**  
Количество уникальных кодов обращений (посещений) для новых пациентов, которые были впервые зарегистрированы в системе

> 🎯 **Определение:** Новый пациент — тот, у кого это первый визит в клинику по данным МИС
>
> 📌 **Примечание:** Позволяет оценить поток первичных обращений

---

> 💡 **Рекомендации по использованию:**
> - **Визит ≠ Услуга:** один визит может включать несколько услуг — учитывайте это при анализе нагрузки
> - **Плановые метрики** доступны только при наличии настроенного плана посещений в системе
> - **Посещений на пациента** — ключевой индикатор лояльности: рост может указывать как на улучшение здоровья (реже ходят), так и на ухудшение (чаще ходят)
> - При расчёте **Посещений в час** учитывайте только фактически отработанные часы по дошедшим пациентам
> - **YTY/MTM** метрики чувствительны к сезонности — сравнивайте корректные периоды
