## <a id="head-1"></a>🔧 # услуг

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Количество услуг |
| Services Count |
| Volume |

**📝 Описание:**  
Общее количество оказанных услуг за выбранный период

> 📌 **Примечание:** Учитываются все услуги, зафиксированные в МИС, независимо от статуса оплаты

---

## <a id="head-2"></a>📅 # услуг прошлого года

**📝 Описание:**  
Количество оказанных услуг в прошлом году за период, аналогичный выбранному

> 💡 **Примечание:** Базовая метрика для расчёта годовой динамики (YTY)

---

## <a id="head-3"></a>🏥 # услуг в кабинетах

**📝 Описание:**  
Количество услуг, оказанных в амбулаторных кабинетах

> ⚙️ **Особенность:** Рассчитывается при учёте клиникой расписания по кабинетам приёма
>
> 📌 **Примечание:** Позволяет оценить загрузку амбулаторного звена отдельно от стационара

---

## <a id="head-4"></a>📊 Среднедневное количество услуг

**📝 Описание:**  
Среднее количество услуг, оказываемых за один день выбранного периода

> 🧮 **Расчёт:** `# услуг / # дней в выбранном периоде`
>
> 🎯 **Интерпретация:** Индикатор средней дневной нагрузки на медицинский персонал

---

## <a id="head-5"></a>📈 YTY услуг, %

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Услуги, YTY% |
| YearToYear, % |
| Динамика услуг, % |

**📝 Описание:**  
Процентное изменение количества оказанных услуг к прошлому году (Year-To-Year)

> 🧮 **Расчёт:** `100 * (# услуг текущий период - # услуг прошлый год) / # услуг прошлый год`

---

## <a id="head-6"></a>🌤️ Ср. дн. #услуг сезона

**📝 Описание:**  
Среднедневное количество услуг для конкретного сезона

> 🧮 **Расчёт:** `Ср. дн. # услуг за сезон / Кол-во дней в сезон`
>
> 💡 **Примечание:** Вычисляются 4 показателя для каждого из сезонов (зима, весна, лето, осень) по данным последнего сезона

---

## <a id="head-7"></a>📉 ∆ ср. дн. #услуг (сезон), %

**📝 Описание:**  
Отклонение текущей среднедневной нагрузки по услугам от сезонной нормы

> 🧮 **Расчёт:** `(Ср.дн. #услуг - ср. дн.# услуг за сезон) / ср. дн.# услуг за сезон * 100`
>
> 💡 **Примечание:** Вычисляются 4 показателя для каждого сезона (зима, весна, лето, осень)
>
> 🎯 **Интерпретация:** Помогает выявить аномалии нагрузки относительно типичного сезона

---

## <a id="head-8"></a>🔗 Услуг на посещение

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Services per Visit |
| Средняя глубина чека по услугам |

**📝 Описание:**  
Среднее количество услуг, которое приходится на одно посещение

> 🧮 **Расчёт:** `# услуг / # посещений`
>
> 🎯 **Интерпретация:** Показывает, насколько комплексно обслуживается пациент за один визит

---

## <a id="head-9"></a>📊 Услуг на посещение, YTY

**📝 Описание:**  
Динамика показателя «Услуг на посещение» к прошлому году

> 🧮 **Расчёт:** `Услуг на посещение в этом году / Услуг на посещение в предыдущем году - 1`
>
> 📌 **Примечание:** Рост показателя может указывать на увеличение комплексности обслуживания или кросс-продажи

---

## <a id="head-10"></a>📈 # услуг продано, YTY

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Изменение количества услуг, абс |
| Services Delta YTY |

**📝 Описание:**  
Абсолютное изменение количества услуг, оказанных по годам

> 🧮 **Расчёт:** `# услуг - # услуг прошлого года`

---

## <a id="head-11"></a>📊 # услуг продано, YTY %

**📝 Описание:**  
Процентное изменение количества оказанных услуг по годам

> 🧮 **Расчёт:** `100 * (# услуг / # услуг прошлого года - 1)`
>
> 🎯 **Интерпретация:** Позволяет оценить темпы роста или снижения объёма услуг

---

## <a id="head-12"></a>🗂️ Группа АВС услуги

**📝 Описание:**  
Классификация услуг по методу ABC-анализа в зависимости от вклада в общую выручку

| Группа | Доля в выручке | Характеристика |
|--------|----------------|----------------|
| **A** | ~80% | Ключевые услуги, обеспечивающие основную выручку |
| **B** | ~15% | Услуги среднего вклада, стабильный спрос |
| **C** | ~5% | Нишевые или редко заказываемые услуги |

> 🎯 **Применение:** Помогает приоритезировать управление ассортиментом и маркетинговые усилия

---

## <a id="head-13"></a>🎯 # услуг, обеспечившие 80% выручки

**📝 Описание:**  
Количество уникальных услуг, которые в совокупности обеспечивают 80% всей выручки

> 🧮 **Расчёт:** Считаются услуги с наибольшей выручкой, начиная с самой доходной, пока не будет достигнут порог 80%
>
> 🎯 **Интерпретация:** Показывает, насколько сконцентрирована выручка вокруг узкого набора услуг

---

## <a id="head-14"></a>📊 % услуг, обеспечившие 80% выручки

**📝 Описание:**  
Доля уникальных услуг (в процентах от общего ассортимента), которые обеспечивают 80% всей выручки

> 🧮 **Расчёт:** `(# услуг, обеспечивших 80% выручки / Общее количество уникальных услуг) * 100`
>
> 🎯 **Интерпретация:** 
> - Низкий % → высокая концентрация выручки на ключевых услугах
> - Высокий % → более равномерное распределение выручки по ассортименту

---

> 💡 **Рекомендации по использованию:**
> - **ABC-анализ** следует пересчитывать регулярно (ежеквартально), так как структура выручки может меняться
> - Показатель **«Услуг на посещение»** помогает оценить эффективность кросс-продаж и комплексных программ
> - При анализе **сезонных метрик** учитывайте эпидемиологическую обстановку и календарные особенности года
> - **Концентрация выручки** (80% от малого числа услуг) — это риск: диверсификация ассортимента повышает устойчивость бизнеса
> - Метрика **# услуг в кабинетах** требует корректной настройки расписания по кабинетам в МИС
