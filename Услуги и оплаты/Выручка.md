## <a id="head-1"></a>💵 Выручка по услугам

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Фактическая выручка, руб |
| Выручка |
| Revenue |

**📝 Описание:**  
Суммарная выручка по оказанным услугам

---

## <a id="head-2"></a>📅 Выручка предыдущий месяц, руб

**📝 Описание:**  
Суммарная выручка по оказанным услугам предыдущего месяца за период, аналогичный выбранному

> 💡 **Примечание:** Также используется для вычисления показателя **MTM** (Month-to-Month)

---

## <a id="head-3"></a>📆 Выручка прошлого года, руб

**📝 Описание:**  
Суммарная выручка по оказанным услугам прошлого года за период, аналогичный выбранному

> 💡 **Примечание:** Также используется для вычисления показателя **YTY** (Year-to-Year)

---

## <a id="head-4"></a>💳 Выручка по оплаченным услугам, руб

**📝 Описание:**  
Суммарная выручка по оказанным услугам, по которым произошла фактическая оплата

> ⚠️ **Важно:** Используется в случаях, когда есть заметная разница во времени между выставлением услуги и ее оплатой (например, при госпитализации)

---

## <a id="head-5"></a>🆕 Выручка новых пациентов

**📝 Описание:**  
Суммарная выручка по оказанным услугам для новых пациентов

> 🎯 **Определение:** Новым пациент считается **только при своём первом визите**

---

## <a id="head-6"></a>📈 % выручки новых

**📝 Описание:**  
Процентная доля выручки, приходящаяся на «новых» пациентов

---

## <a id="head-7"></a>👤 Выручка по первичным пациентам

**📝 Описание:**  
Суммарная выручка по оказанным услугам для первичных пациентов (интервал первичности — 1, 3, 6 или 12 месяцев)

> 📌 **Примечание:** Первичный пациент — пациент, который не был в клинике в течение всего периода первичности

---

## <a id="head-8"></a>📊 Среднедневная выручка

**📝 Описание:**  
Средняя выручка по оказанным услугам в выбранном периоде за 1 день

> ⚙️ **Особенность расчёта:** Учитываются только дни, где есть хотя бы 1 услуга

---

## <a id="head-9"></a>📅 Среднемесячная выручка

**📝 Описание:**  
Средняя выручка по оказанным услугам за 1 месяц

>  **Расчёт:** Под 1 месяцем понимается 365 дней/12 ≈ **30,42 дней**

---

## <a id="head-10"></a>🏥 Выручка в кабинетах

**📝 Описание:**  
Суммарная выручка по услугам, оказанных пациентам в амбулаторных кабинетах

> 🏢 **Примечание:** Рассчитывается при учёте клиникой расписания по кабинетам приёма

---

## <a id="head-11"></a>💸 Выручка по оплатам (Сумма оплат)

**📝 Описание:**  
Сумма оплат, произведённых за период

> ⚠️ **Важно:** В отличие от выручки по услугам, собирается по финансовым транзакциям. Оплаты не всегда можно отнести к конкретной услуге или к конкретному врачу

---

## <a id="head-12"></a>📊 Среднедневная выручка по оплатам

**📝 Описание:**  
Средняя выручка по платежам за 1 день

---

## <a id="head-13"></a>📅 Среднемесячная выручка по оплатам

**📝 Описание:**  
Средняя выручка по платежам за 1 месяц

> 🧮 **Расчёт:** Под 1 месяцем понимается 365 дней/12 ≈ **30,42 дней**

---

## <a id="head-14"></a>🏷️ Скидка, руб

**📝 Описание:**  
Скидка в рублях

---

## <a id="head-15"></a>🏷️ Скидка, %

**📝 Описание:**  
Скидка в % от прайса

---

## <a id="head-16"></a>📉 Выручка, YTY (абс)

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Выручка, YTY |
| Изменение выручки к предыдущему году (абс) |

**📝 Описание:**  
Изменение выручки к предыдущему году (Year-To-Year, YTY) в абсолютном значении

> 🧮 **Расчёт:** `Выручка - Выручка прошлого года`

---

## <a id="head-17"></a>📈 Выручка, YTY%

**📋 Альтернативные наименования:**

| Название | 
|----------|
| YTY выручки, % |
| Выручка, YTY% |
| YearToYear, % |

**📝 Описание:**  
Процентное изменение выручки к предыдущему году (Year-To-Year, YTY)

> 🧮 **Расчёт:** `100 * (выручка - Выручка прошлого года) / Выручка прошлого года`
>
> 💡 **Примечание:** Доступно в измерениях: Группы клиник, Клиники, Специальности, Отделения

---

## <a id="head-18"></a>📅 Выручка, MTM%

**📝 Описание:**  
Процентное изменение выручки от месяца к месяцу (Month-To-Month, MTM)

> 🧮 **Расчёт:** `100 * (выручка - Выручка предыдущий месяц) / Выручка предыдущий месяц`

---

## <a id="head-19"></a>🔄 ∆ факт выручка MTM, руб

**📝 Описание:**  
Абсолютное изменение (delta) фактической выручки месяц к месяцу

> 🧮 **Расчёт:** `Выручка - Выручка за предыдущий мес.`

---

## <a id="head-20"></a>🔮 Прогноз выручки в текущем месяце

**📝 Описание:**  
Апроксимация фактической выручки на период до конца месяца

> 🧮 **Расчёт:** `Кол-во дней в месяце / Кол-во прошедших дней с начала месяца * Выручка с начала месяца`

---

## <a id="head-21"></a>📉 Выручка тренд до конца текущего месяца, руб

**📝 Описание:**  
Тренд до конца месяца вычисляется исходя из стоимости часа врача за последние 30 дней и выставленного расписания до конца месяца

> 🧮 **Расчёт:** `сумма выручки за текущий период - сумма выручки за аналогичный период прошлого года`
>
> ⚠️ **Важно:** Настройка данного показателя кастомизируется под клинику, т.к. нужно исключать направления деятельности, где выручка не коррелирует с плановым расписанием.

---

## <a id="head-22"></a>🌤️ Ср. дн. выручка сезона

**📝 Описание:**  
Среднедневная выручка по услугам для сезона

> 🧮 **Расчёт:** `Выручка за сезон / Кол-во дней в сезон`
>
> 💡 **Примечание:** Вычисляются 4 показателя для каждого из сезона (зима, весна, лето, осень) по данным последнего сезона

---

## <a id="head-23"></a>📊 ∆ ср. дн. выручки (сезон), %

**📝 Описание:**  
Отклонение текущей среднедневной выручки от сезонной

> 🧮 **Расчёт:** `(Ср.дн. выручка - ср. выручка сезона) / Ср.дн. выручка сезона`
>
> 💡 **Примечание:** Вычисляются 4 показателя, для каждого сезона (зима, весна, лето, осень)

---

## <a id="head-24"></a>🎯 Плановая выручка, руб

**📝 Описание:**  
Выручка, внесенная в план. Может быть внесена как на уровне клиники, так и на уровне отделения

> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-25"></a>⚖️ Отклонение от плана по выручке, руб

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Выручка к плану |
| ∆ выручки к плану |
| Отклонение от плана по выручке, руб |

**📝 Описание:**  
Абсолютное отклонение между фактической выручкой и плановой выручкой в денежном выражении

> 🧮 **Расчёт:** `Факт выручка - Плановая выручка`
>
> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-26"></a>📊 Выручка к плану, %

**📋 Альтернативные наименования:**

| Название | 
|----------|
| ∆ выручки к плану % |
| Выручка к плану, % |

**📝 Описание:**  
Отклонение текущей выручки к плану в %

> 🧮 **Расчёт:** `100 * (сумма выручки - сумма плановой выручки) / сумма плановой выручки`
>
> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-27"></a>✅ Исполнение плана по выручке, %

**📝 Описание:**  
На сколько выполняется план в % (отклонение от 100%)

> 🧮 **Расчёт:** `100 * (Фактическая выручка / Плановая выручка - 1)`
>
> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-28"></a>🎯 Выполнение плана, %

**📝 Описание:**  
Процент выполнения плановой выручки (общий процент)

> 🧮 **Расчёт:** `100 * (выручка - плановая выручка) / плановая выручка) + 100`
>
> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-29"></a>📅 ∆ плана к прошлому году, %

**📝 Описание:**  
Процентное изменение плановой выручки относительно выручки прошлого года

> 🧮 **Расчёт:** `100 * (плановая выручка - выручка за предыдущий год) / выручка за предыдущий год`
>
> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-30"></a>🤝 Вклад в прирост, %

**📝 Описание:**  
Доля от процентного изменения выручки к предыдущему году (Year-To-Year, YTY)

> 🧮 **Расчёт:** `100 * (Выручка (клиника/отделение/специальность) - Выручка прошлый год (клиника/отделение/специальность)) / Выручка прошлый год (общая)`
