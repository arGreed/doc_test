## <a id="head-1"></a>👤 # пациентов / # уникальных пациентов

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Количество пациентов |
| Уникальные пациенты |
| Patient Count |

**📝 Описание:**  
Количество пациентов, которым были оказаны услуги за выбранный период

> 🔑 **Определение:** Пациент определяется по уникальному идентификатору из МИС (дубликаты исключаются)

---

## <a id="head-2"></a>📅 # пациентов в предыдущем году

**📝 Описание:**  
Количество пациентов, которым были оказаны услуги в предыдущем году за период, аналогичный выбранному

> 💡 **Примечание:** Используется для вычисления показателя **YTY** (Year-to-Year) по пациентам

---

## <a id="head-3"></a>📆 Уникальных пациентов в прошлом году

**📝 Описание:**  
Количество уникальных пациентов в аналогичном периоде прошлого года

> 📌 **Важно:** Учитываются только уникальные идентификаторы пациентов, без повторений

---

## <a id="head-4"></a>📈 Уникальных пациентов, YTY%

**📝 Описание:**  
Относительное изменение количества уникальных пациентов к прошлому году (Year-To-Year)

> 🧮 **Расчёт:** `100 * (Уникальные пациенты текущий период - Уникальные пациенты прошлый год) / Уникальные пациенты прошлый год`

---

## <a id="head-5"></a>🩺 # новых пациентов по врачу

**📝 Описание:**  
Количество пациентов, впервые пришедших к данному врачу за выбранный период

> ⚠️ **Важно:** 
> - Метрика доступна **только в разрезе врачей**
> - Для клиники это могут быть как новые, так и не новые пациенты (важен именно первый визит к конкретному врачу)

---

## <a id="head-6"></a>👨‍⚕️ # новых у врача

**📝 Описание:**  
Количество новых пациентов у врача за выбранный период

> 📌 **Примечание:** Для клиники пациенты могут быть уже не новыми — учитывается первый визит именно к этому врачу

---

## <a id="head-7"></a>📊 % новых у врача

**📝 Описание:**  
Доля новых пациентов у врача от общего количества пациентов этого врача

> 🧮 **Расчёт:** `100 * (# новых у врача / # всех пациентов у врача)`

---

## <a id="head-8"></a>🏥 # новых для клиники

**📝 Описание:**  
Количество новых пациентов для клиники — пациенты, которые впервые обратились в данную клинику

> 🎯 **Определение:** Новый пациент для клиники — пациент, не имевший ранее визитов в эту клинику (по данным МИС)

---

## <a id="head-9"></a>📈 % новых для клиники

**📝 Описание:**  
Доля новых пациентов для клиники от общего количества пациентов

> 🧮 **Расчёт:** `100 * (# новых для клиники / # всех пациентов клиники)`

---

## <a id="head-10"></a>💰 % выручки новых пациентов

**📝 Описание:**  
Доля выручки, которую принесли новые пациенты, от общей выручки

> 🧮 **Расчёт:** `выручка новых пациентов / выручка * 100`
>
> 🎯 **Интерпретация:** Показывает, насколько клиника зависит от притока новых пациентов в финансовом выражении

---

## <a id="head-11"></a>📊 % новых пациентов по врачу

**📝 Описание:**  
Доля новых пациентов от общего количества пациентов данного врача

> 🧮 **Расчёт:** `# новых пациентов по врачу / # пациентов для данного врача * 100`
>
> ⚠️ **Важно:** Метрика доступна **только в разрезе врачей**

---

## <a id="head-12"></a>🔄 # повторных пациентов

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Количество повторных пациентов |
| Возвращающиеся пациенты |

**📝 Описание:**  
Количество пациентов, пришедших в клинику не первый раз

> 🧮 **Расчёт:** `# пациентов - # новых пациентов`

---

## <a id="head-13"></a>🩺 # повторных пациентов по врачу

**📝 Описание:**  
Количество пациентов, пришедших к данному врачу не впервые

> 🧮 **Расчёт:** `# пациентов у врача - # новых пациентов по врачу`
>
> 📌 **Примечание:** Учитывается история посещений именно этого врача

---

## <a id="head-14"></a>📅 # первичных пациентов (1, 3, 6, 12 мес)

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Первичные пациенты (интервал) |
| Пациенты в периоде первичности |

**📝 Описание:**  
Количество пациентов, которым были оказаны услуги, пришедших впервые в клинику за выбранный интервал времени (период первичности)

> ⚙️ **Настройка:** Обычно в клинике устанавливается единый интервал (1, 3, 6 или 12 месяцев), в течение которого пациент считается первичным
>
> 📌 **Примечание:** Применяется одна из метрик в зависимости от настроек клиники

---

## <a id="head-15"></a>📊 % повторных / % лояльных пациентов

**📋 Альтернативные наименования:**

| Название | 
|----------|
| Доля повторных пациентов |
| % лояльных пациентов |
| Retention Rate |

**📝 Описание:**  
Доля повторных (лояльных) пациентов от общего количества пациентов

> 🧮 **Расчёт:** `# повторных пациентов / # пациентов * 100`
>
> 🎯 **Интерпретация:** Показатель удержания пациентов и лояльности к клинике

---

## <a id="head-16"></a>🩺 % повторных / % лояльных по врачу

**📝 Описание:**  
Доля повторных пациентов у конкретного врача от общего количества его пациентов

> 🧮 **Расчёт:** `# повторных пациентов по врачу / # пациентов для данного врача * 100`
>
> ⚠️ **Важно:** Метрика доступна **только в разрезе врачей**

---

## <a id="head-17"></a>🔗 Пациентов на услугу

**📝 Описание:**  
Среднее количество пациентов, которым была оказана данная услуга

> 🧮 **Расчёт:** `# пациентов / # услуг`
>
> 📌 **Примечание:** Показывает, насколько услуга массовая или нишевая

---

## <a id="head-18"></a>📈 # пациентов, YTY%

**📝 Описание:**  
Динамика количества пациентов к предыдущему году (Year-To-Year)

> 🧮 **Расчёт:** `(количество пациентов в этом году / количество пациентов в предыдущем году - 1) * 100`

---

## <a id="head-19"></a>📊 # повторных/лояльных пациентов, YTY

**📋 Альтернативные наименования:**

| Название | 
|----------|
| # повторных пациентов YTY |
| # лояльных пациентов YTY |
| Изменение повторных пациентов |

**📝 Описание:**  
Абсолютное изменение количества повторных (лояльных) пациентов по сравнению с предыдущим годом

> 🧮 **Расчёт:** `(Уникальных пациентов - Новых пациентов) - (Уникальных пациентов в предыдущем году - Новых пациентов в предыдущем году)`

---

## <a id="head-20"></a>📈 % лояльных пациентов, YTY

**📝 Описание:**  
Процентное изменение доли повторных (лояльных) пациентов по сравнению с предыдущим годом

> 🧮 **Расчёт:** `100 * (Уникальных пациентов - Новых пациентов) / (Уникальных пациентов в предыдущем году - Новых пациентов в предыдущем году) - 1`

---

## <a id="head-21"></a>❤️ # лояльных пациентов

**📝 Описание:**  
Количество лояльных (повторных) пациентов в определенный период времени

> 🧮 **Расчёт:** `Уникальных пациентов - Новых пациентов`
>
> 🎯 **Определение:** Лояльный пациент — тот, кто уже обращался в клинику ранее и вернулся снова

---

## <a id="head-22"></a>👥 Пациентов на врача

**📝 Описание:**  
Среднее количество пациентов, приходящихся на одного врача за выбранный период

> 🧮 **Расчёт:** `# всех пациентов / # врачей, имевших приёмы`

---

## <a id="head-23"></a>🆕 # новых пациентов

**📝 Описание:**  
Количество новых пациентов за выбранный период (в разрезе всей клиники/сети)

> 🎯 **Определение:** Новым считается пациент при своём **первом визите** в клинику

---

## <a id="head-24"></a>📅 # новых пациентов в предыдущем году

**📝 Описание:**  
Количество новых пациентов за аналогичный период прошлого года

> 💡 **Примечание:** Используется для расчёта динамики привлечения новых пациентов

---

## <a id="head-25"></a>📈 Новых пациентов, YTY%

**📝 Описание:**  
Относительное изменение количества новых пациентов к аналогичному периоду прошлого года

> 🧮 **Расчёт:** `100 * (# новых пациентов текущий период - # новых пациентов прошлый год) / # новых пациентов прошлый год`

> 💡 **Рекомендации по использованию:**
> - При анализе динамики пациентов всегда проверяйте период сравнения (год/месяц)
> - Метрики «новых» и «повторных» пациентов зависят от корректности определения первого визита в МИС
> - При работе с врачебными метриками учитывайте, что «новый для врача» ≠ «новый для клиники»
> - Интервал первичности (1/3/6/12 мес) настраивается глобально для клиники — уточняйте актуальное значение
> - Показатели лояльности (% повторных) — ключевой индикатор качества сервиса и удержания пациентов
