## <a id="head-1"></a>👤 Выручка на пациента (APRC)

**🏷️ Альтернативные наименования:**

| Название |
|----------|
| Average Revenue per Client |
| APRC |
| Ср. чек на пациента |

**📝 Описание:**  
Средняя выручка по услугам, оказанным одному пациенту за выбранный период

---

## <a id="head-2"></a>👨‍⚕️ Выручка на врача

**📝 Описание:**  
Средняя выручка на одного врача за выбранный период

---

## <a id="head-3"></a>💊 Средняя стоимость услуги

**🏷️ Альтернативные наименования:**

| Название |
|----------|
| Средняя стоимость услуги |
| Фактический средний чек, руб |

**📝 Описание:**  
Средняя выручка на одну оказанную услугу

---

## <a id="head-4"></a>📊 Медиана изменения стоимости услуг

**📝 Описание:**  
Медианное значение изменения стоимости услуг за выбранный период

> ℹ️ **Примечание:** Используется для оценки типичного изменения цен, устойчивого к выбросам

---

## <a id="head-5"></a>🚪 Выручка на визит

**🏷️ Альтернативные наименования:**

| Название |
|----------|
| Средний чек визита |

**📝 Описание:**  
Средняя выручка от всех услуг, оказанных пациенту в течение дня посещения клиники

---

## <a id="head-6"></a>🏥 Выручка на посещение

**🏷️ Альтернативные наименования:**

| Название |
|----------|
| Фактический средний чек |
| Средний чек посещения |

**📝 Описание:**  
Средняя выручка от всех услуг, оказанных пациенту в течение одного посещения клиники

---

## <a id="head-7"></a>📋 Плановый средний чек, руб

**📝 Описание:**  
Плановая средняя выручка на одно плановое посещение, рассчитанная на основе планового количества посещений и плановой выручки

> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-8"></a>📈 Ср. чек на посещение, YTY (руб)

**🏷️ Альтернативные наименования:**

| Название |
|----------|
| Ср. чек, YTY, руб |

**📝 Описание:**  
Динамика среднего чека к прошлому году в абсолютном выражении (в рублях)

---

## <a id="head-9"></a>📈 Ср. чек на посещение, YTY

**📝 Описание:**  
Динамика средней выручки на одно посещение относительно предыдущего года (коэффициент)

---

## <a id="head-10"></a>📊 Ср. чек, YTY %

**📝 Описание:**  
Процентное изменение среднего чека по годам (Year-To-Year, YTY)

> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-11"></a>🎯 ∆ ср. чека к плану, %

**📝 Описание:**  
Процентное отклонение фактического среднего чека от планового значения

> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-12"></a>✅ Исполнение плана по ср. чеку, %

**📝 Описание:**  
Процентное выполнение плана по среднему чеку (отношение факта к плану)

> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-13"></a>💰 ∆ ср. чека к плану, руб

**📝 Описание:**  
Абсолютное отклонение среднего чека от плана в рублях

> ⚠️ **Важно:** Может присутствовать только на дашбордах с плановыми показателями

---

## <a id="head-14"></a>🔄 APRC, YTY

**📝 Описание:**  
Динамика к прошлому году показателя APRC (средней выручки на одного пациента)

## 💡 Глоссарий терминов

| Термин | Расшифровка |
|--------|-------------|
| **APRC** | Average Revenue Per Client — средняя выручка на клиента |
| **YTY** | Year-To-Year — сравнение с аналогичным периодом прошлого года |
| **Визит** | День, в который пациент обратился в клинику (может включать несколько услуг) |
| **Посещение** | Факт оказания услуги (может быть несколько посещений в один визит) |
